Grama Urja ⚡
Smart Rural Electricity Management Application

Grama Urja is an Android-based application developed to improve communication between villagers and electricity authorities by providing a simple platform to report and monitor electricity-related issues in rural areas.

The application enables users to report problems such as:

Power cuts
Damaged streetlights
Transformer failures
Electrical maintenance issues

Authorities can track complaints, update statuses in real time, and ensure faster resolution of problems.

Features 🚀
User Authentication using Firebase
Real-time Complaint Management
Live Status Updates
Push Notification Alerts
Simple and User-Friendly Interface
Rural-focused High Contrast UI Design
Timestamp-based Update Tracking
Tech Stack 🛠️
Kotlin
Android Studio
Firebase Authentication
Firebase Realtime Database
Jetpack Compose
Material Design Components
System Workflow 📱
User logs into the application
User reports electricity-related issue
Complaint is stored in Firebase Database
Authorities review and update complaint status
Users receive status updates and notifications
Project Objective 🎯

The main objective of Grama Urja is to provide a digital solution for efficient rural electricity issue management, improving transparency, communication, and response time between citizens and electricity departments.
